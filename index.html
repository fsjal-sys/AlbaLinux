Alba Linux is a portable live Linux distribution with built-in privacy tools. Simply flash the Alba iso to any portable storage medium such as a USB flash drive, and you will be able to boot into the Alba environment on any machine.

Turn any flash drive into your own personal traveling operating system.

Recommended System Requirements:

  • Storage Medium Size >= 3GB
  • Minimum RAM: 256MB
  • Internet connection

Included Tools

Anonsurf
Anonsurf is a tool that routes all of the system's traffic through the Tor network automatically on system startup.
Anonsurf configures packet IP packet filter rules using Tor IPTables.

Tor Browser
The Tor Browser is a tried and true web browser that anonymizes your browser web traffic.
The Tor Browser protects the information regarding the your browsing activity from your ISP.

XFCE Desktop Environment
The Alba system uses the XFCE desktop environment.
XFCE is a lightweight desktop environment that is both fast and low on system resources.
